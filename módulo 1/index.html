2. Nova interface

Conceito
Pensando na melhor flexibilidade na instalação das alturas, foi pensado em um cooktop. A interface foi projetada como um painel tátil. O fogão possui quatro bocas de gás com controles independentes e em altura acessível para cadeirantes e pessoas com nanismo ou gigantismo. Os botões são elevados e texturizados para feedback tátil, com braille integrado para deficientes visuais. Um display LED de alto contraste exibe status. Há alertas sonoros e vibratórios para deficientes auditivos.

  • Cores: azul para frio/seguro e amarelo para quente (visível para daltonistas).
  • Formas: botões circulares e ergonômicos para fácil alcance.
  • Sons: alertas em tons variados.
  • Textos: grandes e em linguagem simples.

3. Manual de utilização do fogão

Explicação dos Botões e Comandos

  • Botão ON (verde, elevado, com braille "Ligar"): Liga todo o fogão. Pressionar por 2 segundos para ativar.
  • Botão OFF (vermelho, elevado, com braille "Desligar"): Desliga todo o fogão. Pressionar por 3 segundos para desligar.
  • Botões por boca (azul, circulares e texturizados): Um para cada boca (1 a 4). Pressionar para acender e girar levemente para ajustar chama entre baixa, média e alta.
  • Barra de temperatura (azul e tátil): Deslize o dedo para ajustar intensidade. Vibra em níveis: 1 vibração (baixa), 3 (alta).

Instruções gerais

  • Para acender uma boca: Pressionar o botão ON (será emitido o bipe de confirmação). Ao pressionar o botão da boca, uma chama azul vai acender e uma luz LED pisca verde. Para o caso de deficiência visual, poderá sentir pela vibração e ouvir "Boca 1 ligada".
  • Para ajustar temperatura: Deslize o dedo na barra de temperatura para a posição desejada (esquerda: baixa; direita: alta). O display mostra nível (1-5) em números grandes e braille. Uma voz irá dizer a temperatura atual.
  • Para desligar: Pressionar o botão OFF para apagar. O fogão desliga automaticamente após 60 min.
